# 1. Database architecture

> New entries at the top, dated `YYYY-MM-DD`, initialled.

## 2026-07-08 — MS
Adopted the three-layer model: canonical `data.csv`, machine schema
`datapackage.json` (Frictionless Table Schema), generated `codebook.md`. Rationale:
plain text is diffable and versionable; a spreadsheet is a binary blob that defeats
`git blame`. Codebook is generated, not hand-maintained, to keep one source of truth.
Schema kept minimal and additive-extensible pending archival work.
