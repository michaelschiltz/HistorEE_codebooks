# 5. Quality and nature of the sources

## 2026-07-19 — MS

`money_exchange` (両替 / ryōgae, e.g. `mon_to_monme_via_board_rate` on `CR-0005`) conversions are priced from the posted ryōgae board rate for the transaction's date, not a modern reference table. The board rate is itself a primary source: its own folio/call number is recorded in `source_ref` alongside the transaction it prices, so the rate is as auditable as the transaction it converts. Confidence on these rows reflects the board rate's own legibility, independently of the transaction entry's.

## 2026-07-08 — MS

Per-source assessment: archive, series, language, legibility, known lacunae, and
the confidence it warrants. Feeds the `confidence` coding in the data.
